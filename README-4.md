# خلفاء راشدین — Khulafa-e-Rashideen

**Live Site:** [asma-umar-dev.github.io/My-Website-The-Khulafa-e-Rashideen](https://asma-umar-dev.github.io/My-Website-The-Khulafa-e-Rashideen/)

A bilingual (Urdu/English) educational website dedicated to the four Rightly Guided Caliphs (Khulafa-e-Rashideen) — Hazrat Abu Bakr Siddiq, Hazrat Umar Farooq, Hazrat Uthman Ghani, and Hazrat Ali al-Murtaza (رضی اللہ عنہم). Covers their biographies, virtues, key events, sacrifices, and includes an interactive quiz/test section.

## ✨ Features

- **Home** — Introduction with the Hadith on following the Sunnah of the Prophet ﷺ and the rightly guided Caliphs, plus a summary card for each of the four Caliphs (title, caliphate period, and how each passed away)
- **Caliphs (خلفاء)** — Detailed profile for each Caliph including:
  - Birth name, birth year/place, tribe/family, caliphate duration
  - Bilingual (Urdu + English) biography and introduction
  - Major achievements during their caliphate
  - Death/martyrdom details with Hijri and Gregorian dates
- **Events (واقعات)** — Landmark historical events with supporting Quranic verses, each in Urdu with English translation:
  - The Cave of Thawr (Abu Bakr)
  - Umar's acceptance of Islam
  - Compilation of the Quran (Uthman)
  - Laylat al-Mabit — Ali sleeping in the Prophet's bed
  - The Battle of Khaybar — Ali and the flag
- **Virtues (فضائل)** — Curated Hadith and sayings of the Prophet ﷺ praising each Caliph, bilingual
- **Sacrifice (قربانی)** — Detailed accounts of each Caliph's sacrifices and martyrdom/passing, bilingual
- **Quiz & Test (کوئز / امتحان)** — Interactive 30-question quiz and a separate 30-question historical test, with live score tracking (Correct / Wrong / Total)
- **Bilingual Toggle** — Urdu content paired with English (🇬🇧 EN) translations throughout
- **Dark/Light Mode** — 🌙 theme toggle

## 🛠️ Tech Stack

- HTML5, CSS3, JavaScript (single-page static app)
- RTL (right-to-left) Urdu typography support
- Hosted via **GitHub Pages**

## 📁 Getting Started

Clone the repo and open `index.html` directly, or serve it locally:

```bash
git clone https://github.com/asma-umar-dev/My-Website-The-Khulafa-e-Rashideen.git
cd My-Website-The-Khulafa-e-Rashideen
# then open index.html in your browser
# or serve locally:
npx serve .
```
